# OSINT Social Network Analyzer

An OSINT-based intelligence tool designed to detect relationships between users by analyzing text similarity and identifying coordinated activity across social networks.

---

## 🚀 Features

- Text similarity analysis using TF-IDF  
- Detection of coordinated behavior between users  
- Graph-based network visualization  
- Community (cluster) detection  

---

## 🛠 Technologies

- Python  
- NetworkX  
- Scikit-learn  
- Matplotlib  

---
